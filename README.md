# ASTRO-SQUAD: LEGACY

**ASTRO-SQUAD: LEGACY** is a high-octane, squad-based space shooter game built entirely in vanilla JavaScript and HTML5 Canvas. Command the Azure Unit and lead your Green Vanguard squad through dangerous sectors to eliminate the alien threat.



## Features
* **Squad Mechanics:** Control your main ship while AI-controlled wingmen provide fire support.
* **Upgrade System:** Use credits earned from fallen enemies to enhance your Hull Plating and Plasma Damage in the Hangar.
* **Dynamic Difficulty:** Face increasingly difficult waves of alien invaders as you progress through sectors.
* **Epic Boss Fights:** Test your skills against the Mutation Overlord every 5 sectors.
* **Retro Visuals:** Clean, vector-style graphics with a parallax starfield background and a sci-fi HUD.

## Built With
* **HTML5 Canvas** - For high-performance 2D rendering.
* **Vanilla JavaScript** - Object-oriented entity management without external libraries.
* **CSS3** - Futuristic UI design using the Orbitron web font.

## How to Play
1.  Open `index.html` in any modern web browser.
2.  **Move:** Your ship follows the mouse cursor.
3.  **Shoot:** The Azure Unit and your squad fire automatically.
4.  **Shop:** Visit "The Hangar" from the main menu to upgrade your stats using credits.
